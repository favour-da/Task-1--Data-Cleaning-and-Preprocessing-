# Task-1--Data-Cleaning-and-Preprocessing-
## Data Analyst Internship
## Dataset Used
Medical Appointment No Shows 
## Tools Used
Phyton(Pandas)
## Objective
To clean and prepare a raw datasetby handling common data quality issues
## Steps Perfomed 
1. Loaded the dataset and explored its structure
2. Checked for missing values- none found
3. Checked for duplicates rows- none found
4. Standardized the Gender column- stripped spaces and converted to uppercase
5. Converted ScheduledDay and AppointmentDay columns from text to datetime format
6. Checked for invalid ages- removed rows with age below 0
7. Saved the clean dataset as medical_appointments_cleaned.csv
## Files in this Repository 
- Medical Appointment no shows.ipynb- Jupyter Notebook with all code
- medical_appointments_cleaned.csv- Final cleaned dataset
## Outcome 
A clean, structured dataset ready for analysis.
